# Code Standards

Language

TypeScript only.

Avoid JavaScript.

Components

Functional Components only.

Naming

PascalCase

Components

camelCase

Variables

UPPER_CASE

Constants

Folder Structure

Feature based.

Maximum Component Length

250 lines.

Maximum Function Length

40 lines.

Reusable Components

Mandatory.

Avoid duplicate code.

Use Server Components when possible.

Client Components only if required.

Never use inline styles.

Always use Tailwind.

Use Zod validation.

Use React Hook Form.

Strict TypeScript.

ESLint must pass.

Prettier formatting required.
