# UI Rules

Always use generous whitespace.

Never clutter the interface.

Cards must have rounded corners.

Animations must feel natural.

No sudden transitions.

No bright colors.

No heavy gradients.

Avoid unnecessary borders.

Use blur carefully.

Use glass effect only where necessary.

Use subtle shadows.

Maximum content width

1280px

Use 8px spacing system.

Accessibility is mandatory.

Dark mode first class citizen.

Motion should enhance UX.

Never distract the user.

Every page should feel calm.
