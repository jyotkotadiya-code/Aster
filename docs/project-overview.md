# Project Overview

## Project Name

Aster

## Tagline

A calmer way to learn.

## Mission

Aster is a web application designed for students who study from YouTube.

Instead of using the distracting YouTube interface, Aster provides a peaceful, minimal and aesthetic learning environment while helping students maintain healthy study habits.

---

## Vision

Learning should feel calm, focused and healthy.

Aster removes unnecessary distractions while encouraging regular breaks, hydration, eye exercises and mindful studying.

---

## Target Users

- Students
- Self learners
- College students
- Competitive exam aspirants
- Online course learners

---

## Core Features

- Import YouTube playlists
- Save YouTube videos
- Watch inside Aster
- Health reminders
- Pomodoro Timer
- Eye Exercise
- Water Reminder
- Break Reminder
- Study Analytics
- Shared Playlists
- Friend Collaboration
- Beautiful Dashboard
- Dark & Light Theme
